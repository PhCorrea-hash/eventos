O que é o Sonart?

Sonart é um site desenvolvido por mim onde é possível encontrar vários eventos de diferentes estilos em diferentes cidades. O intuito do site é demonstrar minhas habilidades de design e programação, mas também para passar uma mensagem que eu acredito ser de vital importância: como o som e a arte (Sonart) podem influenciar nossa vidas. Tendo esse ideal em mente, resolvi desenvolver um site onde as pessoas poderiam achar o evento que mais se enquadra so seu estilo e também onde poderiam compartilhar esses eventos com mais pessoas.

Desenvolvimento do site

Para o desenvolvimento do site foi necessário a utilização de diversas linguagens de programação e ferramentas de design

    Programação

  • HTML
  • CSS
  • JavaScript
  • Python
  • Django
  • PostgreSQL

    Design

  • Photoshop
  • Canva
  • Figma

Para renderizar o site utilizei o site Render, que permite a utilização de um domínio gratuito e também disponibiliza um banco de dados, o qual usei para armazenar as informações de eventos, grupos, mensagens e usuários. Por ser um serviço gratuito, o site demora a carregar e fica fora do ar depois de um certo tempo sem acessos. Além disso o banco de dados é limitado, porém o Render serviu muito bem a este projeto, que tem como principal objetivo demonstrar minhas habilidades.

Objetivos do Sonart

  • Desempenho e Escabilidade
  • Segurança e Privacidade
  • Interface intuitiva e responsiva
  • Automatização de Processos
  • Integração com APIs externas
  • Desenvolvimento Modular e Escalável
  • Real-time Communication
  • Testes e Qualidade de Código
  • Design Clean e Funcional
